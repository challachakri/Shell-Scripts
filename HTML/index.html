Build Tools: Ant,Maven

Pre-requisite Software for Ant,Maven and Gradle are:Java
Default File-name of Build Tools are:
Ant:build.xml
Maven:pom.xml
Gradle:build.gradle

Code Quality Analyzer: Sonarqube.

Default's for Sonarqube Server is:
Port Number:9000
Credentials: Username=admin,Password=admin
Command to start/stop the Sonarqube Server:"sh sonar.sh start/stop(Linux)"

Default's for Nexus:

Port Number:8081

Webserver: Tomcat

Default Port Number for Tomcat:8080
Important Folders:
bin-Used for "Starting" and "Stopping" the Server.
conf:
=>server.xml-Used for "Changing the Port Number" of Tomcat Server.
=>tomcat-users.xml:Used for "Creating the Users" which will help while logging into the Server.
logs:Used for Checking whether "Tomcat Server is Started or not".
webapps:Used for "Deploying the Applications".

Application Server: Jboss/Wildfly

Default Port Number for Jboss/Wildfly Server:8080
Important Folders:
bin-Used for "Starting" and "Stopping" the Server.
standalone:
configuration=>standalone.xml-Used for changing the "port number" of Wildfly Server.
deployments-Used for "Deploying the Applications".

Continous Integration Tool: Jenkins

Pre-requisite Software for Jenkins is "Java"
Default Port Number for Jenkins:8080
Command to run Jenkins in Default Port Number Mode:"java -jar jenkins.war"
Command to run Jenkins in Custom Port Number Mode:"java -jar jenkins.war --httpPort=xxxx"
Default Path for Jenkins home directory:"/var/lib/jenkins"

Containerization Platform: Docker

Command to install Docker in Ubuntu Operating System:"curl -fsSL get.docker.com | /bin/bash"
For Knowing the version of Docker:"docker -v | docker --version | docker version"
Important Topics in Docker are:
Dockerfile
Docker images
Docker Containers
Docker Volumes
Docker Networks
Docker Compose
Docker Swarm

Cloud Platform: AWS
